## 🚀 Installation

### Requirements

- Python 3.6+
- Git
- Linux (Kali Linux / Parrot OS / Ubuntu recommended)

### Quick Install

```bash
git clone https://github.com/yourusername/hackingtool.git
cd hackingtool
python3 install.py
```

### Docker

```bash
docker build -t hackingtool .
docker run -it hackingtool
```

## 🖥️ Usage

```bash
python3 hackingtool.py
```

Navigate through the menu using the number keys to select the desired tool category, then select the specific tool you want to use.
